                      DEPLOYMENT INSTRUCTIONS

PREREQUISITES
1. Ensure .NET Framework 4.7.2 is installed
2. Windows 7 SP1 or later
3. Administrator access (optional, for some features)

INSTALLATION STEPS
1. Copy application files to destination folder
2. Run Web Browser.exe
3. Application creates Downloads folder automatically
4. All necessary files generated on first run

CONFIGURATION
Settings are stored in:
  - Registry: HKCU\Software\[Application]
  - Files: [App Folder]\settings.xml
Changes persist across sessions

UNINSTALLATION
1. Delete application folder
2. Downloads folder can be kept separately
3. Registry entries can be cleaned up manually

                      KNOWN LIMITATIONS
?????????????????????????????????????????????????????????????????????????????

CURRENT LIMITATIONS (V1.0)
1. Uses IE engine - some modern sites may not render
2. Max 3 concurrent downloads
3. No bandwidth throttling
4. No download resume from partial
5. No automatic scheduling
6. No proxy configuration

WORKAROUNDS
1. Modern sites: Use external browser option
2. More downloads: Queue/wait for completion
3. Throttling: Start fewer downloads
4. Resume: Delete and restart
5. Scheduling: Manual control
6. Proxy: Use system proxy
